# vue-admin
> 
 + vue全家桶脚手架，只是一个初始化架构,集成了
 +  vue
 + vuex
 + vue-router
 + element-ui
 + axios
 + vue-i18n
 + rest 用于重置样式
## 初始化
```
yarn install
```

### 开发
```
yarn serve
```

### 部署
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
